# The Telos — Akien's charter (2026-07-14)

Layer 0 of the Cairn intention taxonomy (see `cairn/MAP.md`). Everything in
Cairn traces up to one of these six; anything that can't, doesn't belong.

1. Demonstrate inference compilation.
2. Allow Akien to build tools that help with writing and remember things.
3. Share those tools with others.
4. Build something that thinks the way Akien does.
5. Build something that is self-improving.
6. Build something that makes life suck less for everybody.
