# I-cairn-claude-md — the session brief that cannot rot

*Declared 2026-07-14. Drafted by CC (Fable) from the founding conversation +
MAP.md (upstream evidence); ratified by Akien (challenge run: no human
objection; gate run found one amendment — the owner line — which stands).
Owner-writer: the builder (CC); changes ratified by Akien. Migrates into the
intention envelope when that schema lands (MAP.md Q6).*

CLAUDE.md is the first file every mind reads in Cairn, and it stays true and
small by construction. It contains exactly three kinds of content: the Laws;
pointers to charters (never restatements); and rules not yet enforceable by
physics, each carrying the ticket that will make it physics and retire it
from the file. It contains no negative history — no rule of the form "there
is no X anymore" — because Cairn minds carry no contaminated context to
override. Every line traces to a Law or the Telos. It has its own charter and
stands the /challenge gate on a regular cadence, and it fits in a budget a
mind can absorb in one read (~2k tokens — a tunable, not a law).

**Falsifier:** a /challenge run finds a line that restates instead of points,
can't trace, references the quarry, or shouts a rule that enforcement already
covers — any hit is a red.

**Horizon:** re-challenged at every addition; the IOU class shrinks
monotonically as the tester grows.
